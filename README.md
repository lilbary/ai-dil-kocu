# 🤖 AI Dil Koçu (AI Language Coach)
Gemini AI destekli, İngilizce cümlelerinizi analiz eden ve gramer geri bildirimi veren modern bir mobil uygulama.

## 🚀 Proje Hakkında
Bu proje, kullanıcıların yazdığı İngilizce cümleleri yapay zeka ile analiz ederek hataları düzeltir ve daha doğal alternatifler sunar. **React Native** ve **FastAPI** kullanılarak geliştirilmiş tam kapsamlı (Full-Stack) bir projedir.

## ✨ Temel Özellikler
- 🔐 **JWT Authentication:** Güvenli kullanıcı kayıt ve giriş sistemi.
- 🧠 **AI Gramer Analizi:** Google Gemini API ile detaylı geri bildirim.
- 📜 **Geçmiş Yönetimi:** Analiz edilen tüm cümlelerin PostgreSQL'de saklanması ve yönetilmesi (CRUD).
- ☁️ **Cloud Database:** Neon DB entegrasyonu ile veriler her an yanınızda.

## 🛠️ Kullanılan Teknolojiler
- **Frontend:** React Native (Expo), Axios
- **Backend:** Python, FastAPI, SQLAlchemy
- **Yapay Zeka:** Google Gemini Pro
- **Veritabanı:** Neon (PostgreSQL)

## ⚙️ Kurulum
1. Backend klasöründe `.env` oluşturup API anahtarlarınızı girin.
2. `pip install -r requirements.txt` komutuyla bağımlılıkları yükleyin.
3. `npx expo start` ile mobil uygulamayı ayağa kaldırın.
